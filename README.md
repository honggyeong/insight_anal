# 대구 지역 장애인휠체어 급속충전기 접근성 분석 탐구 프로젝트

## 📋 프로젝트 개요

**프로젝트명**: 공공데이터포털을 활용한 대구 지역 장애인휠체어 급속충전기와 교통약자 이동지원센터 종합 접근성 분석

**연구자**: 홍경영 (고등학교 3학년)

**연구 기간**: 2024년 7월 ~ 2025년 1월

**연구 목적**: 실제 공공데이터를 활용하여 대구 지역의 장애인휠체어 이용자 접근성 현황을 분석하고, 이를 바탕으로 실용적인 모바일 애플리케이션 개발의 기초 자료를 제공

---

## 🎯 연구 동기 및 배경

### 연구 동기
장애인휠체어 이용자들이 일상생활에서 가장 어려워하는 것 중 하나는 전동휠체어 배터리 소모로 인한 이동 제약입니다. 특히 대구 지역에서 급속충전기와 이동지원센터의 분포 현황과 접근성을 체계적으로 분석하여, 실제 이용자들이 필요로 하는 정보를 제공하는 것이 이 연구의 핵심 동기입니다.

### 연구의 의의
- **실용성**: 실제 공공데이터를 활용한 객관적 분석
- **지역성**: 대구 지역 특화 분석으로 지역 맞춤 정책 제언
- **확장성**: 분석 결과를 모바일 앱 개발에 직접 활용
- **사회적 가치**: 장애인 편의시설 개선을 위한 실증적 근거 제공

## 🔬 분석 방법론

### 1. 탐색적 데이터 분석 (EDA)

- **분포 분석**: 지리적 분포, 기능별 분포 패턴
- **시각화**: 히스토그램, 박스플롯, 지도 시각화

### 2. 종합 접근성 점수 산정
```python
# 접근성 점수 계산 공식
accessibility_score = (
    charging_stations * 10 +           # 급속충전기 수 (가중치: 10)
    support_centers * 100 +            # 이동지원센터 수 (가중치: 100)
    avg_concurrent_users * 50 +        # 평균 동시사용가능대수 (가중치: 50)
    (24h_ratio * 100) +                # 24시간 운영 비율 (가중치: 100)
    (air_inflation_ratio * 50) +       # 공기주입 기능 비율 (가중치: 50)
    (phone_charging_ratio * 30)        # 휴대전화충전 기능 비율 (가중치: 30)
)
```

### 3. 통계적 분석
- **t-검정**: 도시-농촌 지역 간 접근성 차이 검증
- **상관분석**: 기능별 비율과 접근성 점수 간의 관계
- **군집분석**: 유사한 접근성 패턴을 가진 구별 그룹화

### 4. 공간적 분석
- **Folium 지도 시각화**: 인터랙티브 지도에 시설 위치 표시
- **밀도 분석**: 구별 시설 밀도 계산 및 색상 구분
- **접근성 지도**: 종합 접근성 점수를 색상으로 표현
